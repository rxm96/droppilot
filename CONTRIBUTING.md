# Contributing

Thanks for your interest in contributing to DropPilot. This doc is intentionally short and focused on local dev and releases.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Releases

Releases are published via Git tags and GitHub Releases. Use the scripts below to bump versions and push tags:

- `npm run release:patch`
- `npm run release:minor`
- `npm run release:major`
